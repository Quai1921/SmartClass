Basic Card Styling:

  • cardBorderWidth - Controls the border width of audio cards (0-10px)
  • cardBorderRadius - Controls the border radius of audio cards (0-30px)

  • cardHoverShadow - Box shadow on hover (various presets)

  • cardCorrectBackgroundColor - Background color when answer is correct
  • cardCorrectBorderColor - Border color when answer is correct
  • cardIncorrectBackgroundColor - Background color when answer is incorrect
